【刷题营 Windows 版】

1. 双击 ShuaTiYing.exe 启动，浏览器会自动打开 http://127.0.0.1:8787
   （保持黑色窗口开着，关掉它就是退出程序）

2. AI 功能（问答题参考答案 / AI 判卷）需要配置 API Key：
   首页 →「⚙️ AI 模型」→「✏️ 配置」→ 云端栏填入你的 DeepSeek API Key → 保存
   - Key 在 https://platform.deepseek.com 注册获取
   - 默认已配好 DeepSeek 官方接口地址，也可以换成任何 OpenAI 兼容服务
   - 不配 Key 也能正常刷选择题，只是问答题没有 AI 帮手

3. 导入新题：首页「📥 导入资料」拖入 docx 即可
   - 选择题格式：1.题干 / A.选项 / 【答案】ABC / 【解析】…
   - 问答题格式：1.问：题干 / （1）小问 /（可选）答：参考答案

4. 你的进度、错题、收藏都存在 data 文件夹里，备份/换电脑拷走它即可。

若杀毒软件误报：PyInstaller 打包的 Python 程序常见误报，添加信任即可；
不放心可在 GitHub 仓库 jimes0813/shuati 查看全部源码。
